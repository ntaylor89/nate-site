## Random Code

```javascript
  const getName = (first, last) => (`${first} ${last}`)

  const name = [ 'Nate', 'Taylor' ]
  console.log(getName(...name))
```

```ruby
  getName = ->(first_name, last_name) { "#{first_name} #{last_name}" }
  name = ['Nate', 'Taylor' ]
  puts getName.call(*name)
```
